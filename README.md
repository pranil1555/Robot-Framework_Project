 # Introduction
This repository contains automation test cases written using Robot Framework. The project demonstrates:

Web automation using SeleniumLibrary.
API testing using RequestsLibrary.
Data-driven testing with reusable keywords.
These test cases showcase best practices in automation testing and are easy to extend or integrate into CI/CD pipelines.

# Features
Web Automation: Automates browser interactions like form submissions, file uploads, and navigation.
API Testing: Validates RESTful APIs for GET, POST, and other methods.
Data-Driven Testing: Runs test cases with multiple data sets stored in Excel files.
Reusable Keywords: Modular approach for creating scalable test scripts.

# Setup and Installation
Prerequisites

Install Robot Framework and required libraries:

pip install robotframework
pip install robotframework-seleniumlibrary
pip install robotframework-requests
pip install robotframework-datadriver
